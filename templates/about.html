RoverOS: Differential Drive Kinematics Analysis

A rigorous computational framework for mobile robot dynamics and position estimation.

Kinematics Foundation

RoverOS implements the complete differential drive kinematics model, which derives from first principles of rigid body motion in the plane (SE(2) transformation group).

The robot's state vector S(t) = [x(t) y(t) θ(t) v(t) ω(t)]T evolves according to the kinematic equations:

dx/dt = v · cos(θ)
dy/dt = v · sin(θ)
dθ/dt = ω

Where x,y represent Cartesian coordinates, θ is the orientation angle, v is linear velocity, and ω is angular velocity. The kinematic mapping employs a half-speed steering frame to account for the inherent nonlinearity inherent in differential drive systems.

Control Pipeline Architecture

The ROS-agnostic control architecture implements an asynchronous client-server paradigm using:

  • Flask-based WebSocket abstraction layer for HTTP communication
  • Vectorized numerical integration using explicit Euler method with variable time stepping
  • Bounded state propagation with collision detection and response algorithms

The system operates at a fixed latency of approximately 16ms, achieving real-time performance through pre-computed trigonometric optimizations.

Collision Avoidance Dynamics

Boundary constraint enforcement utilizes geometric collision detection in robot-centered coordinates, implemented via Hausdorff boundary checking for the rectangular environment:

if ‖rnext‖ > ‖rbounds‖:
  rclamped = rbounds - |rbounds - rnext|
  v = 0 (reflective dissipation)

The collision response mechanism instantaneously attenuates linear momentum while preserving angular momentum components, ensuring conservation of angular momentum in impact scenarios.
